# Object Detection and Tracking using YOLOv8

This project is developed as part of the CodeAlpha Artificial Intelligence Internship.

The application performs real-time object detection using a webcam and the YOLOv8 pre-trained model. It identifies and classifies various objects such as people, bottles, laptops, mobile phones, books, and more by drawing bounding boxes and labels around detected objects.

## Features
- Real-time object detection
- Webcam integration using OpenCV
- YOLOv8 pre-trained model
- Automatic object classification
- Bounding boxes and confidence scores
- Fast and accurate detection

## Technologies Used
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- Computer Vision
